# Setup project

## Run yarn
```bash
yarn
```

## Run project
```bash
yarn android
```

## Update branch
```bash
git fetch
```
## Cập nhật code mới
```bash
git add .
```

## Tạo commit
```bash
git commit -m "feat: ..."
```
